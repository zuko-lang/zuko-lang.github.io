Basic Data types


Okay so now we will discuss some basic datatypes used in plutonium. Every programming language provides you some basic data types(integer,float etc) and so does plutonium.
Following are the datatypes used in plutonium:

Integer


An integer in plutonium is same as an integer in mathematics. It is a whole number with a + or – sign. Integer values in plutonium are written the same way as they are in mathematics. But in computer we have limited memory so the integer type has two subtypes 32 bit Integer and 64 bit Integer which consume 32 bits and 64 bits of memory respectively. You don’t need to worry about these as these are handled by the Interpreter. If the entered integer cannot fit into 32 bits then interpreter will treat it as a 64 bit integer.

Float


Floats in plutonium are double precision floating point numbers. 45.56 is a float.

Byte


A byte in plutonium is represented by the prefix 0x. A byte is written as hexadecimal because it’s shorter to write. For example, the byte 11111111 in Plutonium is written as 0xff where ff is the hex equivalent of 11111111 and 0x is just the prefix. Another example would be 0x64

String


Strings are just alphanumeric data(any text). For example, in hello world program,the “hello world!” was a string. Double quotes around any text make it a string. For example, “abcd” is another string. Note that the quotes are not a part of the data they just represent a string. If you simply write abcd without any quotes the interpreter will think of it as an identifier(variable or function name). Strings are immutable in plutonium which means that you cannot change any character of them via indexing.

Boolean


Booleans are values either true or false.

nil


nil means nothing. It is used to show that there is nothing present(no data).


The following program prints constant values of all the basic data types covered so far. 

println(-10)
println(100)
println(2.253)
println(0xfa)
println("this is a string")
println(true)
println(false)
println(nil)


Output:
-10
100
2.253
0xfa
this is a string
true
false
nil
