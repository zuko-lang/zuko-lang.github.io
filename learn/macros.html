Macros


Macro is a piece of code that is replaced by the interpreter while it is translating the code. Macros are like constant values that you want to use in your program. Macros in Zuko begin with a ‘@’ .For example @INT_MAX is a macro representing the max value of 32 bit integer. Similarly @pi is a macro for value of pi. Wherever in your program you write @pi it gets replaced by 3.14 something by the interpreter.

Following is the list of macros available in Zuko.

NameDescription
@INT_MAXMax value of 32bit integer
@INT_MINMin value of 32bit integer
@FLT_MAXMax value of float
@FLT_MINMin value of float
@INT64_MAXMax value of 64bit integer
@INT64_MINMin value of 64bit integer
@piValue of PI
@eValue of e
@SEEK_CURRUsed with fseek
