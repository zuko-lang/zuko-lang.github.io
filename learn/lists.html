Lists


Lists are used to store more than one value i.e a collection or array of values. Lists are denoted by square brackets [] around them and the elements of list are comma separated. The elements of a list can be of different types. To access an element of list you use the index operator []. The index is the position of the element in list starting from 0. For example, list[0] means 0th element of list, 0 is the index here. In case of lists,the type of index can only be a positive integer.

Example: 
println([1,3,5])
var list1 = [10,20,30]
println(list1)
list1.push(40)
println(list1)
list1.pop()
var list2 = [40,50]
list1.insert(len(list1),list2)
println(list1)


Output
[1,3,5]
[10,20,30]
[10,20,30,40]
[10,20,30,40,50]


  • len() is a built-in function which gives the length of strings or lists.
  • List objects also have some member functions.
  • The push(elem) member function is used to push an element at the end of a list.
  • The pop() member function pops the last element and returns it.
  • The insert(pos,elem) member function inserts an element into the list at pos index.
  • The erase(pos) member function erases element at pos index from the list.
  • The erase(startpos,endpos) member function erases elements from the list starting from startpos index to endpost index.

Always remember that lists in zuko are passed by reference. This means that when you assign a list variable to another variable they both begin to point to the same memory location so changing the list via one variable changes the list in memory.
