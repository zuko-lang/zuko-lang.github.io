Dictionary

A dictionary is a data structure in which each value is mapped to a unique key and that key is used to access that value.
Consider the following code:

var car = {"model":2012,"brand": "Toyota","Owner": "Owner Name"}
println(car["model"])
println(car["brand"])
println(car["Owner"])

In the above example keys are strings but you can use any type as a key.
