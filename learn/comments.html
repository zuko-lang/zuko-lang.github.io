Comments


Comments are not a part of the program they help anyone reading your code understand itâ€™s logic. Comments in plutonium begin with a # character. Any line beginning with a # is considered a comment.

Example:
print("Enter n: ")
var n = int(input())
var sum = n*(n+1)/2 # use formula for sum of first n natural numbers
println("sum = ",sum)

Output:
Enter n: 3
sum = 6


The input() is another built-in function. It takes a string input from user and returns it.
The int() built-in function converts a string consisting of digits only to an integer type and returns it.
The comment in the above program is for the reader only and it is completely ignored by plutonium.
