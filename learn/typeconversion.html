Type Conversion


Conversion between types is known as type conversion. It may be implicit or explicit.

Explicit Type Conversion

In explicit type conversion you explicitly specify the type you want to convert to. For example the built-in function int() converts a float or string to an integer. Using int() is an example of explicit type conversion. There are other builtin functions as well for converting to other types as well. These will be discussed in later section.

Example: 
var i = "450"
var j = 4.55
println(int(i))
println(int(j))

Output: 
450
4

In the above code, string “450” is converted to an int, float 4.55 is converted to int and both are printed. When a float is converted to integer using int() function, the float is truncated and returned as integer.

Implicit Type Conversion

Implicit type conversion occurs when you use operands of different types in an expression.
For example, if you add a float and integer what should be the type of the result?
This is an example of implicit type conversion. The interpreter automatically converts types to give a result. Implicit type conversion is done according to some rules.
Following are implicit type conversion rules:

1) If a float occurs in an expression then interpreter will try to give a float result. For example, 5/2 gives 2 but 5.0/2 will give 2.5 as a result.

2) The comparison operators (<,>,<=,>=) also perform type conversion. But the result is a boolean. For example, 5 < 5.5 will return false. The interpreter promotes lhs 5 to a float to perform the comparison. However the == and != are the only comparison operators that do not perform any type conversion if the types don't match they return false and true respectively.

Example: 
println(1/2)
println(1/2.0)
println(1 == 1.0)

Output: 
0
0.5
false
