Loops


Loops are used to repeat a set of statements a given number of times or until a condition becomes false. Zuko provides 4 types of loops(while , dowhile ,for and foreach)

while loop

This loop is used to repeat some statements until a condition becomes false.

Syntax

while(condition)
{
    #statements
}


Example

var k = 1
while(k<=15)
{
    if(k%3==0 and k%5==0)
    println("FizzBuzz")
    else if(k%3==0)
    println("Fizz")
    else if(k%5==0)
    println("Buzz")
    else
    println(k)
    k = k + 1 #increment k
}

dowhile loop is same as while loop. The only difference is that the condition is not checked for the first time in dowhile loop. To get a dowhile loop you just replace the while keyword with dowhile keyword in the syntax mentioned above

for loop

This is a rather simple one. For loop takes a starting value,ending value and increment. Then it iterates from start to end, adding the given increment to loop control variable each time. In the example below i is the loop control variable.

Example
for(var i=0 to 10 step 2)
{
    println(i)
}
In the above example, start is 0, end is 10 and increment is 2. So the above code prints integers 0,2,4,6,8 and 10. If the increment is 1, you can omit the step 1 part. Plutonium will assume the increment to be 1 by default. The above loop starts from value 0 and ends at value 10 but if you want to loop from 10 to 0 with step of 2, then specify 10 as start,0 as end,step as 2 and use dto keyword instead of to keyword. dto stands for 'down to'. When you use this keyword instead of to, the value specified after step is decremented from loop control variable.
