bignum module


This module provides the following functionalities:


Classes


class bignum

Description: A class to represent bignumbers

__construct__(val)
Description: Constructor
Parameters:
val: A string having digits only.
Returns: nil

__smallerthan__(rhs)
Description: Less than operator overloaded
Parameters:
rhs: A bignumber object.
Returns: true or false

__smallerthaneq__(rhs)
Description: Less than or equal to operator overloaded
Parameters:
rhs: A bignumber object.
Returns: true of false

__greaterthan__(rhs)
Description: Greater than operator overloaded
Parameters:
rhs: A bignumber object.
Returns: true or false

__greaterthaneq__(rhs)
Description: Greater than or equal to operator overloaded
Parameters:
rhs: A bignumber object.
Returns: true or false

__add__(rhs)
Description: Addition operator overloaded
Parameters:
rhs: A bignumber object.
Returns: a new bignumber object with value self + rhs

__sub__(rhs)
Description: Subtraction operator overloaded
Parameters:
rhs: A bignumber object.
Returns: a new bignumber object with value self - rhs

__mul__(rhs)
Description: Multplication operator overloaded
Parameters:
rhs: A bignumber object.
Returns: a new bignumber object with value self * rhs

__div__(rhs)
Description: Division operator overloaded
Parameters:
rhs: A bignumber object.
Returns: a new bignumber object with value self / rhs

__eq__(rhs)
Description: Equality operator overloaded
Parameters:
rhs: A bignumber object.
Returns: true or false

__noteq__(rhs)
Description: Inequality operator overloaded
Parameters:
rhs: A bignumber object.
Returns: true or false

strval()
Description: Returns value of bignumer as a string
Returns: a string



Example 



import bignum

var x = bignum.bignum("1245")
var y = bignum.bignum("4567")

var result = x*y

println(result)
println(result.strval())
