Variables


Variables are memory locations used to store some value during the program execution. As the name suggests the value a variable is storing can change. What types of values can a variable store? All the basic types we discussed in previous topic. Variables store computational results during program execution. Variables should have unique names otherwise a Name Error would occur.

Following is the syntax to declare a variable:

var name = val

Where val can be a constant (integer, float, string, byte, bool) or any expression. Or you can also assign another variable’s value to it. Once a variable is declared you can not declare another variable with the same name. The name of a variable must be unique. A variable name must be alphanumeric and it can also include underscores '_'. The variable name must not begin with a digit.


After declaration of a variable you can change it’s value using the assignment operator ‘=’

Consider the following code:
var a = 10
var b = 30
var c = a
c = 0
var sum = a+b+c
println("sum = ",sum)


Output: 
sum = 40

You can assign a new value to a variable using it's previous value. For example, if i is a variable having value of 1 you can increment i by assigning new value of i+1 to it.

var i=1
i = i+1
println(i)

Output:
2


As Plutonium is a dynamically typed language so a variable can store any type of value. If it was initialized with an integer later in the program you can assign a value of any other type to it.

Example:
var myVar = 65535
println(myVar)
myVar = "I'm a string"
println(myVar)

Output: 
65535
I'm a string
