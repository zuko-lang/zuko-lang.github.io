mariadb module


This module allows you to connect to a mariadb server. It provides the following functionalities:


Classes


class result

Description: Class to represent query results

fetch_row()
Description: fetches one row from the result set and returns it
Returns: a list having string elements

num_rows()
Description: tells the number of rows in the result set
Returns: an integer

num_fields()
Description: tells the number of fields( or columns) in the result set
Returns: an integer

__del__()
Description: object deletor
Returns: nil

class connection

Description: Class to represent a connection to the server

store_result()
Description: Stores the result of connection.
Returns: a result object (see result class)

query(sqlcommand)
Description: Executes a sql command/query
Parameters:
sqlcommand: a string
Returns: nil (use store_result to get results)

close()
Description: Closes the connection
Returns: nil

__del__()
Description: object deletor
Returns: nil

Functions

real_connect(host, username, password, dbname, port = 3306)
Description: connects to a server and returns a connection object
Parameters:
host: hostname (a string)
username: username (a string)
password: password (a string)
dbname: database name(a string)
port = 3306: Connection port (an integer). Defaults to 3306
Returns: a connection object if successful otherwise throws an Error
