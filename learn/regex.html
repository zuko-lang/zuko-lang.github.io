regex module


This module provides the following functionalities:


Functions

search(str, patt)
Description: searches a string using given regex pattern
Parameters:
str: a string to search within
patt: a regex pattern
Returns: a list of string parts that match the regex

match(str, patt)
Description: Checks if a string matches a regex pattern
Parameters:
str: the string to check
patt: the regex pattern
Returns: true or false

replace(str, patt)
Description: replaces parts of string that match regex pattern
Parameters:
str: the string
patt: the regex pattern
Returns: a new string
